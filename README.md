# Pro Designer Landing Page

A sleek and professional landing page template for designers or design agencies, built with **React**, styled with **Tailwind CSS**, and enhanced with **AOS** (Animate On Scroll) animations.

---

## 🛠 Tech Stack

- React.js  
- Tailwind CSS (utility-first styling)  
- AOS – Animate On Scroll (for scroll-triggered animations)  
- Vercel (hosting / deployment)

---

## ✨ Features

- Fully responsive design (desktop / tablet / mobile)  
- Smooth scroll-triggered animations via AOS  
- Clean layout sections: hero, about, services, portfolio, contact / call-to-action  
- Hover and transition effects  
- Easy customization: colors, content, spacing
